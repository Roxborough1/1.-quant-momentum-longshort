
# Quant Momentum Long–Short Strategy

### Barry Cox Price-Momentum Factor Portfolio  
**Quantitative Finance Final Project**

This repository contains a complete implementation and analysis of a Barry Cox price-momentum long–short equity strategy using a universe of 120 large-cap U.S. stocks approximating the holdings of SPY. The project includes:

- Data retrieval for all assets  
- Computation of five lagged Barry Cox price-momentum factors  
- Monthly composite z-score ranking  
- Long/short basket construction  
- Equal-weight & inverse-volatility–weighted (optimized) portfolios  
- Five-year backtest with monthly rebalancing  
- Performance comparison vs SPY  
- Complete LaTeX report + figures
